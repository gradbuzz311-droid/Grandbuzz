Build a complete contributor-based blog platform admin system for "GradBuzz".

-----------------------------------
🧠 PRODUCT VISION
-----------------------------------

GradBuzz is a curated student knowledge platform.

Goal:
- Only verified professionals (developers, professors, placed students) can write blogs
- Maintain HIGH QUALITY content
- Not an open blogging platform

Model:
- Closed contributor system (permission-based writing)
- Admin-controlled publishing

-----------------------------------
👥 USER ROLES
-----------------------------------

1. Admin
- Full access
- Approves contributors
- Reviews and publishes posts

2. Contributor (Verified User)
- Can write posts
- Can save drafts
- Cannot publish directly
- Must submit for approval

3. Reader
- Public user
- Can only read content

-----------------------------------
🔁 CORE WORKFLOW
-----------------------------------

1. User applies to become contributor
2. Admin reviews application
3. Admin approves or rejects
4. Approved user becomes Contributor
5. Contributor writes post
6. Submits for review
7. Admin publishes
