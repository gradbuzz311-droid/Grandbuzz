Design a modern, clean admin panel UI for a blog platform called "GradBuzz".

IMPORTANT:
- Do NOT show "GradBuzz Admin" text in the header.
- The logo itself already includes both icon + text (GradBuzz).
- Keep UI minimal, sharp, and product-grade (inspired by Notion, Linear, YouTube Studio).

-----------------------------------

-----------------------------------

- Background: #F5F0E8 (Cream)

- Borders: #E8E0D5

Typography:
- Headings: Syne Bold

- Accent: Fraunces Italic (only where needed)

Style:
- Flat UI

- No heavy shadows
